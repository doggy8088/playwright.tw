歡迎

歡迎來到 Playwright 社群。我們很高興你在這裡。在我們的社群部分，看看我們的影片部分，觀看有關會議演講、直播、精選影片和發行影片的影片。

技術傳教士​

查看我們的 技術傳教士頁面 來看看那些建立 Playwright 內容並與社群分享的了不起的人們。

GitHub​

我們喜歡星星，所以請確保你在 GitHub 上給我們加星。

請建立一個問題如下:

  • 錯誤回報
  • 功能請求
  • 回報回歸
  • 回報安全漏洞

貢獻​

請查看我們的貢獻指南 如果你想為 Playwright 做出貢獻。

社群 Discord​

加入我們的社群 Discord Server 來與其他使用 Playwright 的開發者聯繫，在我們的'help-playwright'論壇中提問，分享你的文章和影片，並參加 Playwright 舞台上的現場活動。

Community LinkedIn​

加入我們的 LinkedIn 社群，與其他開發者聯繫並聽取有關 Playwright 的最新消息。

堆疊 Overflow​

閱讀這些標記為 playwright 的現有問題或提出你自己的問題！

YouTube​

請查看我們的 YouTube channel 以獲取入門系列、精選影片和新版本發佈。

部落格​

請關注我們在 dev.to 上的 Blog，以獲取 Playwright 的官方文章。

新聞​

如需有關 Playwright 的最新消息，請關注 Twitter 上的 @playwrightweb。

Playwright 訓練​

了解如何在 Microsoft Learn 上建構您的第一個端到端測試與 Playwright。
